# workload reconcile

```text
Trigger reconciliation/sync for the root Flux kustomization or root ArgoCD application.

Usage:
  ksail workload reconcile [flags]

Flags:
  -h, --help               help for reconcile
      --timeout duration   timeout for waiting for reconciliation to complete (overrides config timeout)

Global Flags:
      --timing   Show per-activity timing output
```
