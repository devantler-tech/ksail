Trigger reconciliation/sync and wait for completion. For Flux, waits for the OCIRepository (if applicable) and all Kustomizations in the Flux namespace to become ready. For ArgoCD, waits for the root application.

Usage:
  ksail workload reconcile [flags]

Flags:
      --timeout duration   timeout for waiting for reconciliation to complete (overrides config timeout)

Global Flags:
      --benchmark   Show per-activity benchmark output
