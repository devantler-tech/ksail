Option '--config' is required.

Description:
  Create a K8s cluster

Usage:
  up [<name>] [options]

Arguments:
  <name>

Options:
  -c, --config <config> (REQUIRED)                           Path to the cluster configuration file
  -m, --manifests <manifests>                                path to the manifests directory [default: ./k8s]
  -fkp, --flux-kustomization-path <flux-kustomization-path>  Path to the flux kustomization directory
  -t, --timeout <timeout>                                    The timeout in seconds to wait for each kustomization to become ready. Defaults to 300 seconds. [default: 300]
  -ns, --no-sops                                             Disable SOPS [default: True]
  -ng, --no-gitops                                           Disable GitOps [default: True]
  --version                                                  Show version information
  -?, -h, --help                                             Show help and usage information
