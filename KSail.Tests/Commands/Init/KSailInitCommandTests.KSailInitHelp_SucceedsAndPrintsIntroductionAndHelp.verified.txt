Description:
  Initialize a cluster

Usage:
  init [options]

Options:
  -n, --name <name>                 The name of the cluster. [default: ksail-default]
  -c, --components                  Generate components to reduce duplication. [default: False]
  -d, --distribution <K3d|Kind>     The distribution to use for the cluster. [default: Kind]
  -hr, --helm-releases              Generate Helm releases for Traefik, Cert-Manager, and PodInfo. [default: False]
  -o, --output <output>             Location to place the generated cluster output. [default: ./]
  -s, --sops                        Enable SOPS support. [default: False]
  -t, --template <Advanced|Simple>  The template to use for the initialized cluster. [default: Simple]
  --version                         Show version information
  -?, -h, --help                    Show help and usage information
