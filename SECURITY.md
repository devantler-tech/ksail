# Security Policy

## Supported Versions

Only the latest released major version of KSail is supported with security updates.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in KSail, please report it privately.

1. Go to the repositoryâ€™s **Security** tab.
2. Select **Report a vulnerability**.
3. Provide a clear description, steps to reproduce, and (if possible) a minimal proof of concept.

Please do not open public issues for security-sensitive reports.

## Disclosure Process

After you submit a report, maintainers will:

- Acknowledge receipt
- Assess impact and severity
- Coordinate a fix and release
- Credit reporters where appropriate (unless you prefer to remain anonymous)
